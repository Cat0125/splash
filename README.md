# Theming
You can edit file `theme.css` to create your own theme!

**Here is a brief description of the variables:**
- `--primary`: The primary color for elements that require emphasis.
- `--secondary`: Color for elements that do not require special attention.
- `--tertiary`: Usually used for background.
- `--primary-active`: The primary color for the elements that the cursor is hovering over.
- `--secondary-active`: The secondary color for the elements that the cursor is hovering over.
- `--translucent-active`: The background color for the semi-transparent buttons the cursor is hovering over.
- `--red`: I have no idea what it is.
- `--border-color`: Color for the borders.
- `--theme-background`: The background color for the body.
- `--theme-text`: Text Color.
- `--segmented-control-active`: The color of the selected item in the segmented control.
- `--primary-dimmer-color`: The background color for overlay.
- `--panel-color`: The background color for elements such as cards and segments.
- `--disabled-opacity`: Opacity for disabled elements.

# Sizes
`.s-*` resizes element to the specified size

**Aviable sizes are:**
- `xxs` (xx-small) - 0.4em
- `xs` (x-small) - 0.6em
- `s` (small) - 0.8em
- `m` (default) - 1em
- `l` (large) - 1.3em
- `xl` (x-large) - 1.6em
- `xxl` (xx-large) - 1.9em
- `xxxl` (xxx-large) - 2.2em
- `xxxxl` (xxxx-large) - 2.5em
